% Abstract
\begin{abstract}
This report explores three algorithms that solve the Traveling Salesman Problem (TSP): Brute Force, Greedy (Nearest Neighbor), and Held-Karp (exact). The study evaluates time performance and solution accuracy on a set of 25 varied tests. 
The aim is to outline the compromises associated with each method, so as to identify the adequate and acceptable solution in real-life use cases. This way, we shall outline the applications of TSP in complex problems with their own constraints related to resource management.

\keywords{Traveling Salesman Problem \and Graphs \and Benchmarking}
\end{abstract}
